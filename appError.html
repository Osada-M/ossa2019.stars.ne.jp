アプリでエラーが出るときの対処法

トップ    アプリ

Python

  • モジュールがインストールされていない

    • アプリ内で使用するモジュールがインストールされていないためエラーが出る

    解決策(anaconda3環境下)：コマンドプロンプトにて
    pip install (モジュール名)
    と打ち、Enterキーを押しモジュールのインストールを行う。

    具体的には、
    アプリを表示する「tkinter」
    pip install tkinter
    パスを取得する「pathlib」
    pip install pathlib
    標準入出力を仮想的に実現する「pyautogui」
    pip install pyautogui
    などをインストールするとよいかも。

  • Pythonのバージョンが古い
    • Python3以前のものを使用しているためエラーが出る。

    解決策：最新のPythonをインストールする。
    バージョンの確認はコマンドプロンプトにて以下を実行する。
    >> python --version
    Python(ここにバージョンが書かれる)
