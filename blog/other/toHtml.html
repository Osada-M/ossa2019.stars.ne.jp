ホームページ内に貼るソースコードの可読性を向上させたい

ソースコードをHTMLに変換し、色付き文字で出力する


アプリ全体のソースコード(GitHub)



Python や JavaScript などのソースコードをそのまま HTML に貼ると、記号が変な風に認識されて文字が崩れたり、全部同じ色なので読みにくくなったりする。 今回はその悩みの(とりあえずの)解決手段となる Python アプリを紹介する。また、このアプリのソースコードはGitHubにて公開している。


まず、< や & などの記号を HTML で表示できるようにするには「<」は「&lt;」、「&」は「&amp;」のように表記しなければならない。

それを考慮し文字列をその形式に変換する関数 func() を書いた。
def func(xs:str='', *, new:str=''):
    def rep(xs):
        xs = xs.replace('&', '&amp;')
        xs = xs.replace('\t', '&nbsp;&nbsp;&nbsp;&nbsp;')
        xs = xs.replace(' ', '&emsp;')
        xs = xs.replace('<', '&lt;')
        xs = xs.replace('>', '&gt;')
        xs = xs.replace('¥', '&yen;')
        xs = xs.replace('←', '&larr;')
        xs = xs.replace('↑', '&uarr;')
        xs = xs.replace('→', '&rarr;')
        xs = xs.replace('↓', '&darr;')
        xs = xs.replace('ˆ', '&circ;')
        xs = xs.replace('˜', '&tilde;')
        xs = xs.replace('—', '&mdash;')
        xs = xs.replace('"', '&quot;')
        xs = xs.replace('\n', '<br>\n')
        return xs

    boo = [ True, True ]
    for x in xs:
        boo[1] = True
        if(boo[0]):
            if(x is ' '):
                new += '&nbsp;'
                boo[1] = False
            else:
                boo[0] = False
        if(x is '\n'):
            boo[0] = True
        if(boo[1]):
            new += rep(x)

    return new

func( 変換したい文字列 ) とすることで HTML に変換することができる。

改行やインデント、記号などが正しく表示されるように校正できた。 これで Python などのソースコードを HTML に変換することは可能になったが、文字が全て同じ色なので読みにくく感じる。 そこで、import や def など特定の文字だけに色をつける。

まず、CSS では文字列の探索が不可能なので HTML に装飾をつけることを考える。


例えば、HTML を
<span class="dec">import</span>
とし、CSS を
.dec{color : orange}
とすれば
import
「import」がオレンジ色で表示される。


これを利用し、Python 、HTML、JavaScript それぞれでよく使う単語を <span class="クラス名">その単語</span> の形式に変換するプログラムを作成する。

また、様々な色で表示させたいので色のクラスを dec0 〜 dec5 までの６種類用意した。

CSS
.dec0{color : orange;}   /* オレンジ */
.dec1{color : cyan;}      /* 水色 */
.dec2{color : #fd7f7f;}  /* 赤 */
.dec3{color : #9dff76;}  /* 黄緑 */
.dec4{color : #fdff83;}  /* 黄色 */
.dec5{color : #e784eb;} /* ピンク */

def rep_py(xs):

    # dec0
    lis0 = ['def', 'return', 'import', 'pass', 'yield', 'lambda', 'raise', 'break', 'continue', 'with', 'while']

        xs = xs.replace(l, '<span class="dec0">'+l+'</span>')
    xs = xs.replace('class ', '<span class="dec0">class</span> ')

    # dec1
    xs = xs.replace('elif', '<span class="dec1">el</span>if')
    xs = xs.replace('if(', '<span class="dec1">if</span>(')
    lis1 = ['else', 'try', 'except']
    lis1_space = ['if', 'for', 'in', 'is', 'as']

        xs = xs.replace(l, '<span class="dec1">'+l+'</span>')
    for l in lis1_space:

    # dec2
    lis2 = ['self', '@', 'event']
    for l in lis2:

    # dec3
    lis3 = ['True', 'False', 'None']

    lis3_kakko = ['not', 'and', 'or']

    # dec5
    # なし

    return xs
