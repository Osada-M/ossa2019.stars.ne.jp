【ベルヌーイの定理】非線形微分方程式を線形に変換する

非線形微分方程式を線形微分方程式に変換する方法であるベルヌーイの定理について解説する

簡単な説明

ベルヌーイの定理の概形は下記の通りである．

\[非線形微分方程式 \ \frac{dy}{dx}=y+y^a \ に対して 関数 \ u=y^{(1-a)} \ と設定し \ u \ と \ x \ の線形微分方程式に変換する． \]


たとえば次のような微分方程式

\[ \frac{dy}{dx}=y+y^2 \]

があった時，y^2が非線形であるため解くことが困難である．
ここでベルヌーイの定理を用いて関数uを次のように定義する．

\[ u=y^{(1-2)} \\ \ =y^{-1} \\ \ =\frac{1}{y} \]

次にuをxで微分し，yを削除する（uとxだけの式にする）．

\[ \frac{du}{dx}=\frac{-y'}{y^2} \\[10px] ここで元の式： \ y'=y+y^2 \ より，\\[10px] \\ \frac{du}{dx}=\frac{-(y+y^2)}{y^2} \\ \ \ \ \ \ \ =\frac{-y-y^2}{y^2} \\ \ \ \ \ \ \ =-\frac{1}{y}-1 \\[10px] ここで \ u=\frac{1}{y} \ より，\\[10px] \\ \frac{du}{dx}=-u-1 \ \ \ \ ・・・① \]

導出した①の式はuとxの線形微分方程式であるので，基本的な微分方程式の解法を用いることができる．

\[ \frac{du}{dx}=-u-1 \\ \ \ \ \ \ \ =-1*(u+1) \\[10px] u+1 \ を左辺に移行させる\\[10px] \\ \frac{1}{u+1}\frac{du}{dx}=-1 \\[20px] 両辺を \ x \ で積分する\\[10px] \\ \int\frac{1}{u+1}\frac{du}{dx}dx=\int-1dx \\ \ \ \ \ \ \int\frac{1}{u+1}du=\int-1dx \\ \ \ \ \ \ \ \ \ \ \ ln|u+1|=-x+C \ \ \ \ ( C : 任意定数 ) \\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ u+1=\pm e^{-x+C} \\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ u+1=Ce^{-x} \\ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ \ u = Ce^{-x}-1 \\[10px] 最後に \ u=\frac{1}{y} \ より，\\[10px] \\ y=\frac{1}{u} \\ \ \ =\frac{1}{Ce^{-x}-1} \ \ \ \ ・・・② \]

最後に導出した②の式が今回の一般解である．
