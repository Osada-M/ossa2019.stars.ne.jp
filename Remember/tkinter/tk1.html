tkinterのインストール

pipが使える場合（Windows、mac共通） :
コマンドプロンプトにて以下の文を実行する。

pip install tkinter


macOS :
ターミナルにて以下の文を実行する。

brew install tcl-tk


yumインストーラを使用している場合 :
以下の文を実行する。

yum install tkinter
